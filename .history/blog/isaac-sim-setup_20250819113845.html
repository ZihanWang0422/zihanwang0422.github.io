June 11, 2025    Zihan Wang    Technical Tutorial

Prerequisites

Official GeForce Drivers NVIDIA

Ubuntu 22.04 NVIDIA Driver/CUDA/cuDNN Installation

CUDA Installation

1. NVIDIA Driver

# Uninstall existing nvidia driver
sudo apt-get --purge remove "*nvidia*"
sudo apt autoremove
# Install specific version driver
sudo add-apt-repository ppa:graphics-drivers/ppa  # Add official ppa source
sudo apt update  # Check package update list
apt list --upgradable  # View upgradable package list
sudo apt upgrade  # Update all upgradable packages
# Download driver
ubuntu-drivers devices    # Ubuntu detects available drivers for nvidia card
sudo apt install nvidia-driver-5xx  # Install graphics driver based on your nvidia card

2. Download CUDA

Download link: https://developer.nvidia.com/cuda-toolkit-archive

3. Switch CUDA Version:

# Check installed cuda versions
ls -l /usr/local | grep cuda
# Open environment variables
vim ~/.bashrc

Add CUDA path configuration to bashrc file:

export CUDA_HOME=/usr/local/cuda-xx.x
export PATH=$CUDA_HOME/bin:$PATH
export LD_LIBRARY_PATH=$CUDA_HOME/lib64:$LD_LIBRARY_PATH

IsaacSim Installation

System Requirements

  • Ubuntu 20.04 or 22.04
  • NVIDIA RTX series graphics card
  • CUDA 11.8 or higher
  • At least 32GB RAM (recommended)
  • At least 50GB available disk space

Download and Installation

1. Visit NVIDIA Omniverse official website to download Omniverse Launcher

2. Install Isaac Sim through Launcher

3. Configure Python environment

# Install dependencies
pip install omni-isaac-sim
pip install opencv-python numpy matplotlib

ROS2 Integration Configuration

Isaac ROS Installation

# Install ROS2 Humble
sudo apt update && sudo apt install curl gnupg lsb-release
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc | sudo apt-key add -
sudo sh -c 'echo "deb [arch=$(dpkg --print-architecture)] http://packages.ros.org/ros2/ubuntu $(lsb_release -cs) main" > /etc/apt/sources.list.d/ros2-latest.list'
sudo apt update
sudo apt install ros-humble-desktop

# Install Isaac ROS
mkdir -p ~/workspaces/isaac_ros-dev/src
cd ~/workspaces/isaac_ros-dev/src
git clone https://github.com/NVIDIA-ISAAC-ROS/isaac_ros_common.git
cd ~/workspaces/isaac_ros-dev
colcon build

SLAM Demonstration

Configure Isaac Sim Scene

1. Start Isaac Sim

2. Load Warehouse scene

3. Add Carter robot

4. Configure sensors (LiDAR, Camera)

Run SLAM

# Terminal 1: Start Isaac Sim ROS Bridge
cd ~/.local/share/ov/pkg/isaac_sim-*
./isaac-sim.sh --/isaac/startup/ros_bridge_extension="omni.isaac.ros2_bridge"

# Terminal 2: Start SLAM
source ~/workspaces/isaac_ros-dev/install/setup.bash
ros2 launch isaac_ros_visual_slam isaac_ros_visual_slam.launch.py

# Terminal 3: Start Navigation
ros2 launch isaac_ros_navigation isaac_ros_navigation.launch.py

常见问题解决

CUDA版本兼容性

确保CUDA版本与Isaac Sim要求一致，避免版本冲突导致的启动失败。

内存不足

Isaac Sim对内存要求较高，建议：

  • 关闭其他占用内存的应用
  • 减少同时加载的模型数量
  • 调整渲染设置降低内存占用

ROS通信问题

检查ROS_DOMAIN_ID设置，确保Isaac Sim和ROS节点在同一网络域中。
