Building PointCloud Reconstruction

Zihan Wang
Tsinghua University


Abstract

We developed a building fine modeling method based on point cloud segmentation and the Iterative Closest Point (ICP) algorithm. Using handheld LiDAR scanners, we generated 3D point clouds of buildings and applied a semantic segmentation model (PointNet++) to improve accuracy and robustness in complex scenes. A geometry-semantic constrained reconstruction algorithm was further designed to restore missing structures and refine building details. The method achieves efficient and high-precision 3D building models, accurately representing details such as walls, roofs, doors, and windows. This approach addresses the challenges of incomplete data and structural complexity in real-world building reconstruction tasks.

Key Features

  • Handheld LiDAR Scanning: Portable and flexible data acquisition for building exteriors and interiors
  • Deep Learning Segmentation: PointNet++ for semantic understanding of building components
  • ICP Registration: Precise alignment of multiple point cloud scans
  • Geometry-Semantic Constraints: Intelligent reconstruction using both geometric and semantic information
  • Missing Structure Recovery: Automatic completion of incomplete building features
  • High-Precision Modeling: Detailed representation of architectural elements

Technical Approach

1. Data Acquisition: We employ handheld LiDAR scanners to capture dense point clouds of building structures. The portable nature of handheld devices allows for comprehensive scanning of both exterior facades and interior spaces, including hard-to-reach areas. Multiple scans from different viewpoints ensure complete coverage of the building geometry.

2. Point Cloud Segmentation: The semantic segmentation module utilizes PointNet++, a deep learning architecture specifically designed for point cloud understanding. The network classifies each point into architectural categories such as walls, roofs, windows, doors, floors, and ceilings. This semantic information is crucial for guiding the subsequent reconstruction process and ensuring structural coherence.

3. Point Cloud Registration: Multiple scans are aligned using the Iterative Closest Point (ICP) algorithm, which finds the optimal rigid transformation to minimize the distance between corresponding points. We employ a coarse-to-fine registration strategy, starting with feature-based initial alignment followed by fine-tuning with ICP. This ensures accurate merging of overlapping scans into a unified coordinate system.

4. Geometry-Semantic Constrained Reconstruction: The reconstruction algorithm combines geometric constraints (planarity, orthogonality, parallelism) with semantic information from the segmentation results. For example, walls are constrained to be planar and vertical, while windows must be embedded within wall surfaces. This constraint-based approach produces structurally valid and visually coherent building models.

5. Missing Structure Recovery: Occlusions and scanning limitations often result in incomplete point clouds. Our method identifies missing regions through semantic analysis and reconstructs them using shape priors and symmetry detection. For instance, if only half of a window is visible, the system can infer and complete the missing portion based on typical window characteristics and building symmetry patterns.

Applications

This building reconstruction technology has wide-ranging applications in:

  • Architecture and Construction: Digital twins for renovation planning and progress monitoring
  • Cultural Heritage Preservation: High-fidelity 3D documentation of historical buildings
  • Urban Planning: Large-scale city modeling and spatial analysis
  • Real Estate: Virtual property tours and accurate floor plan generation
  • Building Information Modeling (BIM): Automated as-built model creation
  • Disaster Assessment: Rapid structural evaluation after natural disasters

Advantages

  • Efficiency: Rapid data collection with handheld devices compared to traditional surveying
  • Accuracy: Millimeter-level precision in reconstructed models
  • Completeness: Intelligent recovery of missing data ensures comprehensive representation
  • Robustness: Deep learning segmentation handles complex scenes and varying conditions
  • Detail Preservation: Fine architectural features are accurately captured and modeled
