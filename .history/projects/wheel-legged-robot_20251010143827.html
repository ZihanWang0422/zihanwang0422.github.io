Abstract

We developed a wheeled-legged balancing robot that combines the advantages of wheeled mobility and legged stability. The system features a custom-designed STM32F103 microcontroller board with embedded software implementing Linear Quadratic Regulator (LQR) and Virtual Model Control (VMC) algorithms. The robot is capable of balancing, load-carrying, and differential-drive locomotion, demonstrating robust performance in various operational scenarios. This project showcases the integration of advanced control theory with practical robotics implementation, resulting in a versatile mobile platform suitable for both research and educational purposes.

  • Load Carrying Capability: Designed to carry payloads while maintaining balance
  • Differential Drive: Enables precise maneuvering and turning capabilities

Technical Details

Control Architecture

The robot employs a hierarchical control structure with LQR at the core for optimal state feedback control. The Virtual Model Control (VMC) approach is used to manage the leg dynamics, treating the legs as virtual springs and dampers. This dual-controller architecture ensures stable balancing while allowing for dynamic motion capabilities.

Hardware Components

The custom STM32F103 board serves as the main computing unit, interfacing with motor drivers, IMU sensors, and encoders. The embedded system runs real-time control loops at high frequencies to maintain balance and execute motion commands. The mechanical design features a compact form factor with efficient power distribution and thermal management.

Applications

This wheeled-legged robot platform can be applied to various scenarios including indoor delivery systems, mobile manipulation tasks, educational robotics, and as a research platform for control algorithm development. The combination of stability, mobility, and load-carrying capability makes it suitable for environments where traditional wheeled robots may face challenges.
