Zhi Xing Robot (智行机器人)

Zihan Wang
Tsinghua University

Abstract

We developed an autonomous mobile car using the Robot Operating System (ROS), integrating comprehensive mapping, navigation, and voice interaction modules for real-world deployment. The system combines simultaneous localization and mapping (SLAM) capabilities with intelligent path planning algorithms to enable autonomous navigation in complex indoor environments. The voice interaction module allows for natural human-robot communication, making the robot accessible and user-friendly. This project demonstrates the successful integration of multiple robotic subsystems into a cohesive autonomous platform suitable for various service and research applications.

Key Features

  • ROS Framework: Built on the Robot Operating System for modular and scalable design
  • SLAM Mapping: Real-time simultaneous localization and mapping capabilities
  • Autonomous Navigation: Intelligent path planning and obstacle avoidance
  • Voice Interaction: Natural language processing for human-robot communication
  • Sensor Fusion: Integration of multiple sensors for robust perception
  • Real-world Deployment: Tested and validated in actual operational environments

Technical Architecture

Software Stack: The robot's software architecture is built on ROS, utilizing packages such as gmapping for SLAM, move_base for navigation, and custom-developed modules for voice interaction. The modular design allows for easy maintenance and future extensions. All modules communicate through ROS topics and services, ensuring seamless integration and real-time performance.

Mapping and Localization: The mapping module employs laser range finders to build 2D occupancy grid maps of the environment. Adaptive Monte Carlo Localization (AMCL) is used for continuous pose estimation, enabling the robot to track its position accurately within the constructed map. This combination ensures reliable navigation even in dynamic environments.

Navigation System: The navigation stack implements both global and local path planning. Global planners compute optimal routes from the current position to the destination, while local planners handle real-time obstacle avoidance and smooth trajectory execution. The Dynamic Window Approach (DWA) is utilized for local planning, ensuring safe and efficient motion.

Voice Interaction: The voice interaction module integrates speech recognition and natural language understanding to interpret user commands. The system can respond to navigation requests, status queries, and operational commands, providing audio feedback through text-to-speech synthesis. This creates an intuitive interface for non-technical users to interact with the robot.

Applications

The Zhi Xing Robot platform is suitable for various applications including indoor delivery services, elderly care assistance, museum tour guidance, warehouse logistics, and educational demonstrations. Its robust navigation capabilities and user-friendly voice interface make it adaptable to numerous service robotics scenarios. The system serves as an excellent platform for research in autonomous mobile robotics and human-robot interaction.
