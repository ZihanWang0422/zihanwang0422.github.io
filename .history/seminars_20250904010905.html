Complete collection of robotics course materials from top universities including Stanford, MIT, CMU, and UC Berkeley. This comprehensive resource covers all major areas of robotics from fundamentals to advanced topics. All notebooks are interactive and include practical implementations, theoretical background, and hands-on exercises.

Covered Topics:
  • Computer Vision: Feature detection, stereo vision, object recognition, deep learning for vision
  • Path Planning: A*, RRT, sampling-based planners, ROS navigation stack
  • SLAM: EKF-SLAM, FastSLAM, visual SLAM, graph optimization, loop closure detection
  • Control Systems: PID control, state-space methods, optimal control, adaptive control
  • Machine Learning: Deep reinforcement learning, policy gradients, robot learning
  • Manipulation: Kinematics, dynamics, grasping, motion planning, force control
  • Perception: Sensor fusion, filtering, probabilistic robotics
  • Navigation: Localization, mapping, autonomous driving, multi-robot systems
